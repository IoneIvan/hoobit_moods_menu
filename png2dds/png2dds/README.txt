# Image Converter

This is a simple image conversion program that converts images to the BC2_UNORM format using the `texconv.exe` tool.

## Installation

To use this program, you need to have the `texconv.exe` tool available. You can find more information and download the tool from the [BinomialLLC/crunch](https://github.com/BinomialLLC/crunch/blob/master/README.md) repository.

## Usage

1. Launch the `ImageConverter.exe` program.
2. Click the "Browse" button or manually enter the path to the folder containing the images you want to convert.
3. Click the "Start Conversion" button to begin the image conversion process.
4. The progress bar will show the progress of the conversion.
5. Once the conversion is complete, a message box will appear indicating the completion.

Note: The output folder will be created in the same directory as the `ImageConverter.exe` program.

## Support

For more information and support regarding the `texconv.exe` tool and image conversion, please refer to the [BinomialLLC/crunch](https://github.com/BinomialLLC/crunch/blob/master/README.md) repository.
